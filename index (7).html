⚠️ Changing your super admin password
Your password is stored as a hash in the code, not in plain text — so anyone viewing the source sees a scrambled string, not your actual password. To change it:

1. On this site, open your browser's developer console (press F12, or right-click → Inspect → Console)
2. Type: await hashPassword('yourNewPassword') and press Enter
3. Copy the long string it returns
4. Open index.html in a text editor, find const SUPER_ADMIN_PASSWORD_HASH, and paste your new hash in as the value
5. Save and re-upload to GitHub

Note: this still runs entirely in the browser (no real server), so it's not unbreakable against a determined attacker — but it's no longer sitting in plain sight either.
